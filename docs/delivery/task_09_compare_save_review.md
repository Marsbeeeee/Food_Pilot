# Task 09: 比较、保存与长期使用链路

- Status: Planned
- Owner: Product / Frontend / Backend
- Last Updated: 2026-04-05
- Source Of Truth: 本文档为 Task 09 当前交付说明
- Depends On: `docs/rules/container_data_model.md`, `docs/system/data_lifecycle.md`

## 目标

让 FoodPilot 从一次性判断工具升级为支持比较、保存、复盘和长期使用的系统。

## 本期范围

1. 设计两商品比较输入和结果结构
2. 优化保存逻辑和容器归档
3. 收敛收藏夹语义
4. 校验卡片标签与落库对象一致
5. 让历史选择与 Profile、Insights 建立弱连接

## 验收

1. 两商品比较结果有明确优先结论和理由
2. 历史记录能区分公共商品与私有商品来源
3. 卡片标签、落库对象、容器归档结果一致
