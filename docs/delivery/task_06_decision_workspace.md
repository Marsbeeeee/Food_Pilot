## 目标

把前端从聊天消息流展示升级为更适合点单决策的工作台式界面。

## 本期范围

1. 设计新的结果卡片层级
2. 区分澄清态、低置信态、成功态 UI
3. 支持套餐分组展示
4. 显示分类标签和保存去向
5. 提供重跑、保存、收藏、继续比较等入口
6. 做移动端适配

## 验收

1. 单屏可扫读输入、结论、风险、建议
2. 标签展示与保存归档一致
3. 收藏、保存、加入分析的准入条件清晰分离
