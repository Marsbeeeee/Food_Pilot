🍽 FoodPilot — AI 驱动的饮食热量与营养分析助手

FoodPilot 是一个以 自然语言交互为核心的 AI Agent 型 Web 应用，旨在帮助用户通过文本对话的方式，快速理解食物的热量与营养结构，并根据个人目标获得更合理的饮食建议。

本项目聚焦于 可解释的营养估算 + 用户友好的交互体验，而非复杂或医疗级计算，适合作为 AI Agent / Web + AI 融合方向的项目展示。

✨ 项目亮点

🤖 对话式 AI Agent 设计
用户通过自然语言描述食物或饮食场景，系统返回结构化、可解释的营养分析结果。

📊 清晰的营养拆解与热量估算
输出包括总热量（kcal）及主要食材的能量构成，避免“黑盒式”回答。

🧠 基于用户画像的个性化建议
支持设置健康目标、每日热量目标、饮食偏好等，用于后续对话中的智能调整。

📒 饮食历史记录（Food Log）
用户可以回看过往的查询与分析结果，形成连续的饮食认知，而非一次性问答。

🎨 温馨、克制的 UI 设计
前端采用现代 Web 技术栈，整体风格偏向健康与生活方式应用，避免技术感压迫用户。

🧩 核心功能模块
1️⃣ Ask FoodPilot（对话分析）

输入任意食物或饮食描述（如“一份鸡肉沙拉加半个牛油果”）

AI 返回：

热量估算（kcal）

食材级别的能量拆解

简要结论与后续建议

2️⃣ My Food Log（饮食记录）

自动保存用户的历史查询

支持回看当时的对话与分析结果

用于形成长期饮食认知，而非精确追踪

3️⃣ Profile（用户画像）

用户可设置：

健康目标（减脂 / 增肌 / 健康 / 表现）

每日热量目标（kcal）

饮食偏好（如高蛋白、无特定限制等）

基础身体与生活方式信息（用于估算合理性）

所有设置均用于生活方式级别的个性化建议，不构成医疗或营养处方。

⚠️ 免责声明

FoodPilot 提供的所有热量与营养信息均为估算结果，基于常见食材与标准份量假设，仅用于信息参考与饮食认知辅助。
本项目不用于医疗诊断或营养处方，用户如有健康相关需求，应咨询专业人士。
